Provides classes for translation to other logical languages in the Sigma knowledge engineering environment version developed at Articulate Software Inc. The core classes are SUMOKBtoTPTPKB to handle translation to TPTP language, THF to handle translation to the higher-order logic language THF, and OWLtranslator for the OWL language.
